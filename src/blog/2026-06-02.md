Major fixes to fog in this update. If you have had issues with it, please try this new version.

## Changes
- Refactor Chest Styles to work with the standalone options
- Reimplement End Portal shaders

## Fixes
- Fix issues with fog density in the Nether and End with Boss Fog
- Fix left-handed Bow holding
- Fix missing item texture issue on 1.21.4
