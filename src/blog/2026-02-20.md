# The Customization Update

![A screenshot of the menu containing all of the options](https://github.com/user-attachments/assets/47e1cb6c-0651-4a44-a9d5-bd8c606af84a)

## New:
- Bamboo Sapling texture
- Player-style Armor Stand
- White Eyes Creaking Heart
- New unique Charcoal texture (with new option to make coals match)
- Customizer via PolyTone (1.21.11+)
- Four option presets built-in (Early Beta, Late Beta, Early Release, Late Release)
- Player CEM to flatten the second layer (PolyTone & EMF)
- Sky Dimension/Aether inspired End option (PolyTone)
- Built-in Bushy Leaves option (PolyTone)
- Overlay CTM option (PolyTone)
- more below...
- Started on 26.1 baby mobs (not done)

## All New Options:
- Bushy Leaves: true, false
- Overlay CTM: disabled, normal, no_tint
- Beta Lighting: true, false
- Disable Flat Items: true, false
- Use Core Shaders: true, false
- Cloud Height: -64, 320
- Bricks: alpha, modern
- Cobblestone: classic, hybrid, modern
- Cuboid Bamboo: true, false
- Fast Grass: true, false
- Flatten Skins: true, false
- Matching Coals: true, false
- Oak Log Tops: true, false
- Shadowed Leaves: true, false
- Silent Chests: true, false
- Solid Items: true, false
- Chest Style: beta, hybrid, modern
- End Style: custom, early_release, modern
- XP Bar Style: hidden, custom, modern
- [WIP] Picture Perfect: true, false

## Changes:
- Oak Log Tops is now a PolyTone option and not the default
- Retextured Bamboo (and new 3D model option)
- Retextured the Deepslate set
- Retextured the Blackstone set
- Retextured the Tuff set
- Retextured Trapped Chest ring
- Make all PolyTone modifiers explicitly targeted
- Removed ResPackOpts support

## Parity:
- Edited lighting on Repeaters and Comparators
- Force nearest neighbor sampling on 1.21.11+ (shader)
- Changed the breaking particles on Grass Blocks

## Fixes:
- Fixed Boats bouncing on land
- Fixed White Eyes not spawning on 1.21.11+ (PolyTone)
- Fixed Crying Obsidian particles not matching the base texture

## Golden Days Alpha:
- New option for Classic Dyes (includes dyed blocks excluding Terracotta) (PolyTone)
- New option for Classic Bricks (PolyTone)
