# The Customization Update - Patch 1

![Water particles coming out the back of the boat](/images/boatparticles.png)

## New
- Wind Charges are now 2D like snowballs (EMF)
- Ghasts ignore lighting and are always glowing (EMF)
- Boats now emit water particles when moving (PolyTone)
- OBJ paralleliped Torch models for NeoForge
- Item rename easter egg for Torches
- Texture and rename Golden Flowers (26.1)

## Changes
- Hoes now make the dirt break sound like in old versions
- Birch log tops now use the oak bark rim
- Leaf Litter is now hidden from maps

## Options
- New option for 2D cross candles like in old versions of Golden Days
- Auto-generated versions of the PolyTone options are now available as standalone packs on GitHub in the actions tab for versions without PolyTone configs (not fully supported)

## Fixes
- Fix Fire model backfaces being visible
- Hide the outer hat layer of Baby Zombie Pigmen
- Fix bat wings being invisible from the back
- Added missing Oak translations
