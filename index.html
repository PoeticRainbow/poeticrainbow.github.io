Golden Days: A Minecraft Time Machine.

Golden Days is a texture pack for Minecraft that sends your game back in time. It changes blocks, items, mobs, and even GUI elements to be inspired by the past.

Golden Days Base

The main pack, meant to be usable in survival gameplay and does not remove any newer features in order to make it functional. Lighting, sounds, textures, and models! Everything you need for an authentic feeling experience. Use this pack on top of Programmer Art in order for it to work.

Golden Days Alpha

The Alpha addon changes a few things, the main one being the iconic Alpha grass foliage color! Use this pack on top of Golden Days Base in order for an amazing Alpha-like experience in Modern Minecraft!



Client Mod Support:

This pack has extended capabilities using some modded features:

- Config Options (Chest Model, Cobblestone texture..., requires PolyTone)

- Custom Particles (found in the Pale Garden..., requires PolyTone)

- Modifiers (Block, Item, Biome, Dimension, Fluid..., requires PolyTone)

- Custom Colors (Dyed Sheep, Text Colors, Fireworks..., requires PolyTone)

- Custom Entity Models (Boats, Zombie Pigmen, Allays, Bats..., requires EMF)


Directly Supported:

PolyTone: Adds many resource pack capabilities. Config Options, Custom Colors, block properties, etc.

Entity Model Features: Gives Golden Days the ability to modify entity models.

Pairs Well:

Nostalgic Tweaks: A mod for Fabric/Forge that adds a whole ton of old Minecraft tweaks! It is really cool!

Moderner Beta: Adds accurate beta and alpha generation back into Minecraft!

Not Supported:

OptiFine: No longer the intended experience as open source replacements are available.

Supported in Older Versions:

Enhanced Block Entities (<1.21.4): Allows block entities to use JSON block models, gives full-block shaded chest models with Golden Days

Colormatic (<1.20.1): Adds OptiFine style Custom Colors support.
